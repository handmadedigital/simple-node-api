# Simple Node.js Api using Express

An easy way to get started with Node.js api.

## Features
- Express
- REST API

## Requirements
- [node & npm](https://nodejs.org/en/)

## Installation
- `git clone git@github.com:handmadedigital/simple-node-api.git`
- `cd simple-node-api`
- `npm install`
- `npm start`

### GET Routes
- visit http://localhost:3000
  - /
  - /shopify
